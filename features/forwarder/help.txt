══════════════ 超级转发器 V3.8 ══════════════
监控源群新内容，支持限制群穿透、相册聚合及定时清洗转发。

📂 [状态查询]
/fw_show_lib     - 预览资源配置 (6xx源/7xx目标/8xx规则/9xx任务)。
/fw_cache_clean  - [编号] 立即清空指定源群的待发缓冲区。

⚙️ [清洗规则定义 - 8xx]
/fw_rule [8xx] [时间] [自定义后缀]
   - 示例: /fw_rule 801 20:00 更多资讯: @MyChannel
   - 逻辑 A: 内容含 # 标签 -> 仅保留 #标签 + 后缀。
   - 逻辑 B: 内容无 # 标签 -> 保留原文样式 + 后缀。

🚀 [任务链路控制 - 9xx]
/fw_task [6xx] [7xx] [8xx] 
   - 绑定源、目、规，启动每日定时任务。
   - 系统 24h 自动监控，并在指定时间批量分发。
/fw_task_del [9xx] 
   - 销毁任务，停止监控并清理相关状态。

📥 [底层资源登记]
/fw_src [6xx] [ChatID] - 登记监控源群 (支持频道/受限群)。
/fw_dst [7xx] [ID1,ID2] - 登记转发目标群组矩阵。

🛡️ [工业级核心特性]
1. [相册聚合]: 自动合并 MediaGroup，延迟等待，拒绝刷屏。
2. [强制穿透]: 遇禁止转发限制，自动启动 copy_message 模式。
3. [样式保留]: 清洗过程不破坏超链接、粗体等 Entity 样式。
4. [深度去重]: 基于内容 MD5 哈希，忽略空格干扰，防止重复入库。
